# 查看一个模块是否有对应的types包

```
npm view <package_name> version

npm view @types*@*

npm vew 
```



# 常用`@types`文件收集

| 名称                                   | 说明         | 备注 |
| -------------------------------------- | ------------ | ---- |
| `@types/node`、`@types/node@<version>` | node 支持    |      |
| `@types/md5`                           |              |      |
| `@types/resolve`                       |              |      |
| `@types/rimraf`                        | 文件删除插件 |      |
| `@types/ejs`                           |              |      |
