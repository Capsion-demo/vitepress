# 自动生成颜色卡片

[官网](https://javisperez.github.io/tailwindcolorshades/?royal-blue=7575FB)

![image-20220310170842240](https://gitee.com/capsion-images/notebook/raw/master/image/202203101708444.png)

一键生成支持 tailwind 的颜色卡片

```js
// tailwind.config.js
'royal-blue': {
    '50': '#f8f8ff', 
    '100': '#f1f1ff', 
    '200': '#ddddfe', 
    '300': '#c8c8fd', 
    '400': '#9e9efc', 
    '500': '#7575FB', 
    '600': '#6969e2', 
    '700': '#5858bc', 
    '800': '#464697', 
    '900': '#39397b'
}
```
